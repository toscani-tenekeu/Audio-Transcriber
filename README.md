# Audio Transcriber

Simple browser-based audio transcription UI that calls OpenAI's speech-to-text API.

Usage
- Open `audio-transcriber.html` in your browser.
- Paste your OpenAI API key in the inputs and select an audio file (<= 25 MB).
- Click "Transcribe" to get the transcript.

Note: The page sends your API key directly to api.openai.com for transcription and does not save the key anywhere.
